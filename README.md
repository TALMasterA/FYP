# FYP (Using Azure API to develop translation function)
 Develop in Android Studio, Android ONLY.
 Azure: Use Azure Recon.
 Google: Recording - Stop - Use Google Recon. (Can be used, but the function has stopped updating)

 Caution:
 Testing APP: Connect the phone and the Computer with the USB
 Enable USB debugging in the phone's Developer Options
 Turn OFF when not testing, or you can not get in your online bank!

 Remember to enter your key in Azure Speech Service (& Google Text-to-Speech API) in build.gradle.kts(:app), and Sync to test.

 If BuildConfig is in red, try sync + Clear & Rebuild, if still red, just run.
